# IT-Support-Ticket

## Setup Instructions

1. **Prerequisites:**

   - Java 17 installed.
   - Maven installed.

2. **Clone the Repository:**

   ```bash
   git clone <repository-url>
   cd IT-Support-Ticket
   ```

3. **Build the Project:**

   ```bash
   mvn clean install
   ```

4. **Run the Application:**
   ```bash
   mvn spring-boot:run
   ```
